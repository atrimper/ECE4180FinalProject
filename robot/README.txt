This directory is for robot obstacle detection, serial communication, and trajectory control code.

Hardware List:
     Robot Chassis (https://www.sparkfun.com/products/13301)
     Mbed NXP LPC1768 (https://www.sparkfun.com/products/9564)
     Pushbutton (https://www.sparkfun.com/products/9190)
     2x Wheel (https://www.sparkfun.com/products/13259)
     2x DC Motor (https://www.sparkfun.com/products/13302)
     H-Bridge Motor Driver (https://www.sparkfun.com/products/14450)
     2x Hall Effect Encoder (https://www.sparkfun.com/products/12629)
     3D-Printed Angle Bracket (../lidarMount.SLDPRT)
     VL53L0X ToF Distance Sensor (https://www.adafruit.com/product/3317)
     LSM9DS1 9DoF IMU (https://www.sparkfun.com/products/13944)
     Battery Holder - Barrel Jack (https://www.sparkfun.com/products/9835)
     DC Barrel Jack Adapter (https://www.sparkfun.com/products/10811)
     SPDT Switch (https://www.sparkfun.com/products/102)

Usage Note: Remove the mbed library in the LSM9DS1_Library_cal library in order to compile.
